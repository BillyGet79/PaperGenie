# PaperGenie开发日志

## 项目实现目标（更新时间：2025-6-14）
- 通过RAG技术实现对论文内容的智能问答
- 将论文内容向量化，建立索引
- 实现论文内容的自动摘要和生成wiki页面
- 提供一个友好的用户界面，方便用户查询和浏览论文内容
